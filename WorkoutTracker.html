Full Stack Workout Tracker

View on GitHub

Full Stack workout tracking application allowing users to log, update, and delete exercises

This application utilizes a RESTful API built with Node.js and Express for the backend, and a React frontend for a seamless user experience.

Implemented a dynamic react frontend with collapsible components for better user experience

Built and connected Flask backend using SQLAlchemy and SQLite for data persistence
